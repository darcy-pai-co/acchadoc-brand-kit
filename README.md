# AcchaDoc Brand Kit

Complete brand identity system for AcchaDoc - India's medical certificate platform.

## 🎯 Quick Start for LLMs

This repository contains everything needed to build AcchaDoc. All design decisions and assets are documented.

**⚠️ CRITICAL: Read `00-CRITICAL-RULES.md` first - especially RULE #1: NO EMOJIS**

## 📦 Contents

- `00-CRITICAL-RULES.md` - Non-negotiable brand rules (READ FIRST)
- `01-color-palette.css` - Cyan, Emerald, Amber color system
- `02-typography.css` - Plus Jakarta Sans + Inter fonts
- `03-logo-concepts.md` - Logo variations and usage
- `04-ui-components.tsx` - React + Tailwind components
- `05-icons-library.tsx` - SVG icon system (NO EMOJIS)
- `06-whatsapp-messages.md` - Message templates
- `07-certificate-design.md` - A4 certificate specifications
- `acchadoc-showcase/` - Live Next.js brand showcase

## 🏥 Brand

**Name:** AcchaDoc (अच्छा = Good/Fine/OK)
**Positioning:** "The friendly neighborhood doctor, now digital"
**Colors:** Cyan #0891B2 (trust) + Emerald #10B981 (success) + Amber #F59E0B (warmth)
**Fonts:** Plus Jakarta Sans (headings) + Inter (body)
**Icons:** SVG only - Never emojis
