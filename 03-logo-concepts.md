# AcchaDoc Logo Concepts

## Logo Concept 1: Wordmark with Medical Symbol (Recommended)

```
  ╔═══╗
  ║ + ║  AcchaDoc
  ╚═══╝
```

**Description:**
- Simple medical cross icon in a rounded square badge (left)
- "AcchaDoc" wordmark in Plus Jakarta Sans Bold
- Color: Cross in Emerald (#10B981), wordmark in Gray 900 (#111827)
- Badge background: Emerald light (#D1FAE5)

**File specs:**
- Logo lockup: Icon + wordmark horizontal
- Minimum size: 120px width
- Clear space: Equal to height of icon on all sides

---

## Logo Concept 2: Checkmark + Wordmark

```
  ✓  AcchaDoc
```

**Description:**
- Checkmark icon (represents "Accha" = approval/affirmative)
- Icon in Emerald (#10B981) circle badge
- Wordmark in Gray 900
- Circle diameter: 32px

**Reasoning:**
- Checkmark = "Accha, approved, all good"
- Simpler than medical cross
- Universal positive symbol

---

## Logo Concept 3: Text-Only Wordmark with Color Split

```
  Accha  Doc
  ─────  ───
  Emerald Cyan
```

**Description:**
- "Accha" in Emerald (#10B981)
- "Doc" in Cyan (#0891B2)
- Plus Jakarta Sans Bold
- No icon, pure typography

**Reasoning:**
- Emphasizes the two-part meaning
- "Accha" (positive, Hindi) + "Doc" (medical, English)
- Clean, scalable, works at any size

---

## Logo Usage Guidelines

### Primary Logo (Concept 1)
Use on: Homepage, app header, business cards, WhatsApp profile

### Text-Only Wordmark (Concept 3)
Use on: Favicon, social media profile, small spaces, watermarks

### Color Variations

**Full Color (Primary)**
- Icon: Emerald on light emerald background
- Text: Gray 900
- Use on: White/light backgrounds

**Reversed (Dark backgrounds)**
- Icon: Emerald
- Text: White
- Use on: Dark gray/photos

**Monochrome**
- All Gray 900
- Use on: Printed certificates, legal documents
