## Message Design Guidelines

### Emoji Usage
- ✓ Use sparingly (1-2 per message)
- ✓ Use for status: ✓ (success), ⚡ (speed), 📱 (action needed)
- ❌ Avoid: Overuse, irrelevant emojis, corporate-y thumbs up

### Formatting
- **Bold** for important info (amounts, next steps)
- *Italics* for labels/categories
- Line breaks for readability
- Numbered lists for steps
- Bullet points for features

### Tone Checklist
- [ ] Sounds like a helpful friend, not a bot?
- [ ] Clear next action stated?
- [ ] No jargon or corporate speak?
- [ ] Would you send this to your own friend?

### Length
- Keep under 150 words per message
- Break long content into multiple messages
- Use "..." to indicate more messages coming
